📖 使用说明
⚡ 基本流程
  • 点击议程名称可直接修改（如"致辞"、"主题演讲"）
  • 计时前用 ± 按钮调整时长，每次 ±5 分钟，范围 5~180 分钟
  • 点击 ▶ 开始计时，计时中可 ⏸ 暂停 / ▶ 继续
  • ↺ 重置：恢复默认时长，回到初始状态
🔔 提醒节点
  • 时长 ≥ 30min → 半程提醒
  • 剩余 15min / 10min / 5min → 分级提醒
  • 超时后 → 超时5min / 10min / 15min 逐级加重
  • 点击 📋 查看所有节点的预计触发时间
🎨 边框颜色含义
剩余 > 15min — 正常
剩余 10~15min — 关注
剩余 5~10min — 紧迫
剩余 < 5min — 危急
已超时 — 闪烁红色
🔊 提醒方式
  • 页面弹窗：居中弹出，必须手动关闭，不可忽略
  • 提醒音：分级递增（半程1声 → 剩余5min4声 → 超时急促警报）
  • 系统通知：需点击 🔔 授权（iOS Safari 不支持浏览器通知）
